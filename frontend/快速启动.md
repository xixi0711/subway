# 地铁智能问答系统 - 快速启动

## 方式一：传统方式（推荐）

### 1. 启动后端
```bash
# 先构建RAG知识库
python test_rag_final.py

# 启动后端服务
python subway_api.py
```

### 2. 启动前端
```bash
# 进入前端目录
cd frontend

# 如果没有安装依赖，先安装
# npm install

# 启动开发服务器
npm run dev
```

### 3. 访问应用
- 后端API: http://localhost:5000
- 前端界面: http://localhost:3000

## 方式二：使用Python服务器运行前端

如果无法使用Node.js，可以用Python服务器：

### 1. 启动后端
```bash
python subway_api.py
```

### 2. 启动前端服务器
```bash
# 进入frontend目录
cd frontend

# 启动Python HTTP服务器
python -m http.server 3000
```

### 3. 访问
浏览器打开 http://localhost:3000

## 方式三：直接打开HTML文件

直接用浏览器打开 `frontend/index.html` 文件（不推荐，部分功能可能受限）

## 注意事项

1. 确保 `subway.db` 数据库文件存在
2. 确保 `.env` 文件中的API密钥配置正确
3. 如果遇到跨域问题，确保后端服务正常运行
