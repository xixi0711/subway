# 地铁智能问答系统 - 前端项目

## 环境要求

- Node.js >= 18.0.0
- npm >= 9.0.0

## 安装步骤

### 1. 安装 Node.js

**方法一：官网下载安装**
1. 访问 https://nodejs.org/
2. 下载 LTS 版本（推荐）
3. 运行安装程序，按照提示完成安装

**方法二：使用包管理器**

Windows (使用 winget):
```powershell
winget install OpenJS.NodeJS.LTS
```

Windows (使用 Chocolatey):
```powershell
choco install nodejs
```

macOS:
```bash
brew install node
```

Linux (Ubuntu/Debian):
```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs
```

### 2. 验证安装

打开终端/命令提示符，输入：

```bash
node --version
npm --version
```

如果显示版本号，说明安装成功。

### 3. 安装前端依赖

```bash
cd frontend
npm install
```

### 4. 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000

### 5. 构建生产版本

```bash
npm run build
```

构建后的文件在 `dist` 目录中。

## 项目结构

```
frontend/
├── index.html          # 入口HTML
├── package.json       # 项目配置
├── vite.config.js    # Vite配置
├── public/            # 静态资源
└── src/               # 源代码
    ├── main.js        # 应用入口
    ├── App.vue        # 根组件
    ├── router/        # 路由配置
    ├── stores/        # 状态管理
    ├── api/           # API调用
    ├── views/         # 页面组件
    ├── components/    # 公共组件
    └── assets/        # 资源文件
```

## 技术栈

- **构建工具**: Vite 5.0
- **框架**: Vue 3.4
- **路由**: Vue Router 4
- **状态管理**: Pinia 2.1
- **UI组件**: Ant Design Vue 4.0
- **HTTP客户端**: Axios
- **CSS预处理器**: Sass

## 主要功能

- 线路管理
- 站点管理
- 时刻表管理
- 刷卡记录查询
- 客流分析
- AI智能问答
- 线网图可视化

## 注意事项

1. 确保后端服务（Python API）运行在 http://localhost:5000
2. 开发环境下，Vite 代理会处理跨域问题
3. 如果遇到问题，先清除浏览器缓存
